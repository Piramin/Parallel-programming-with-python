Для запуска данных программ необходимо установить библиотеку mpi4py.
Gauss_line - реализация метода Гаусса решения системы линейных алгебраических уравнений.
First_project_parallel - первая версия(для ознакомления) реализованного метода Гаусса в параллельном режиме вычислений.
Gauss_Parallel - Финальная версия.

!Параллельные программы тестируются на физической модели задачи о нагреве пластины.
